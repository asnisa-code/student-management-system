# Student Management System

A console-based Student Management System developed using **Python** and **MySQL**. This project performs CRUD operations (Create, Read, Update, Delete) to manage student records efficiently.

## Features
- Add new student records
- View all students
- Update student details
- Delete student records
- MySQL database integration

## Technologies Used
- Python
- MySQL

## How to Run
1. Install MySQL and Python
2. Create database using `database.sql`
3. Install dependencies: `pip install mysql-connector-python`
4. Run the application:
   ```bash
   python student_management.py
